Edu Fit 数学入塾テスト（オンライン版）

このページは、Edu Fit 数学入塾テストをオンラインで解いて、その場で採点できる簡単なアプリです。

  • 全10問・6択式
  • 解き終わったら「採点する」ボタンを押してください。
  • 正解した問題は 薄い緑、不正解は 薄い赤 で表示されます。

第1問

次の計算をしなさい。

\((-2/3)^2 \div (1/6 - 1/2) + 5/3\)

第2問

x = 3, y = -2 のとき、次の式の値を求めなさい。

(2x - 3y) - 3(x - 2y)

第3問

連立方程式を解いたときの x の値を求めなさい。

3x - 2y = 1
5x + 3y = 8

第4問

関数 y = -2x + 5 について、x の変域が -1 ≤ x < 4 のとき、y の変域として正しいものを選択しなさい。

第5問

関数 y = ax^2 のグラフが、点 (3, -18) を通るとき、この関数の x の変域が -2 ≤ x ≤ 1 のときの y の変域として正しいものを選択しなさい。

第6問

1つのサイコロを2回投げるとき、1回目に出た目を a、2回目に出た目を b とする。このとき、点 (a, b) が直線 y = -x + 7 上にある確率を求めなさい。

第7問

底面の半径が 3cm、高さが 4cm の円錐の表面積を求めなさい。ただし、円周率は π とする。

第8問

△ABC において、∠A の二等分線と辺 BC との交点を D とする。AB = 6cm, AC = 4cm, BC = 5cm のとき、線分 BD の長さを求めなさい。

第9問（発展）

関数 y = (1/2)x² のグラフ上に 2 点 A, B があり、A(-4, 8), B(2, 2) である。また、同じグラフ上に点 P(t, (1/2)t²) をとる。ただし、t > 0 とする。

三角形 OAB の面積と三角形 PAB の面積が等しくなるような、実数 t の値として正しいものを次の中から 1 つ選びなさい。

第10問

1辺の長さが 6cm の立方体 ABCD-EFGH がある。辺 AD の中点を M、辺 CD の中点を N とする。この立方体を 3 点 M, N, G を通る平面で切断するとき、頂点 C を含む方の立体の体積を求めなさい。

※ 未回答の問題は不正解として扱われます。
